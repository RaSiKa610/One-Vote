# One Vote (v2.0) 🇮🇳

One Vote is a premium, localized voter education platform designed to align with the ECI SVEEP pedagogical guidelines. It empowers Indian citizens with knowledge about the democratic process, registration, and voting day procedures through a modern, mobile-first interface.

## 🚀 Key Features

### 1. Educational Hubs
- **Voter Categories**: Detailed guides for General, Service, Overseas (NRI), and PwD electors.
- **Electoral Literacy Clubs (ELC)**: Information on school/college ELCs and Chunav Pathshala.
- **Voter Awareness Forums (VAF)**: Dedicated section for awareness in offices and corporates.
- **Media Library**: Curated ECI resources including videos, comics, and literacy games.

### 2. Smart AI Assistant
- **Multilingual Support**: Voice and text assistance in 10 Indian languages.
- **Advanced Knowledge Base**:
  - Current Chief Ministers and ruling parties of all states.
  - Indian political history since 1947.
  - Contributions of major national leaders.
  - Real-time answers on voting fundamentals (NOTA, EVM, MCC).

### 3. Localized Data Analytics
- **Historical Trends**: Interactive charts showing voter turnout since 2004.
- **Participation Metrics**: Gender-wise and age-group distribution.
- **Electorate Growth**: Visualizing the expansion of the Indian electorate.
- **State-wise Turnout**: Comparative participation data for various states.
- **Full Localization**: All chart labels and data keys are translated into the user's chosen language.

### 4. Interactive Tools
- **Civic Knowledge Quiz**: Test your understanding of the election process with instant feedback and explanations.
- **Pre-Vote Checklist**: A prioritized task list to ensure you're ready for election day.
- **Polling Booth Finder**: Integrated search for booth locations (via official ECI links).

## 🛠️ Technology Stack
- **Frontend**: React (Vite), React Router, Lucide Icons.
- **State Management**: React Context API (Language, User).
- **Backend/Auth**: Firebase Authentication & Firestore.
- **Data Visualization**: Recharts.
- **PWA**: Offline-enabled with Service Workers.
- **Localization**: Custom i18n system supporting 10 languages (Hindi, Marathi, Gujarati, Bengali, Tamil, Telugu, Kannada, Malayalam, Punjabi, English).
